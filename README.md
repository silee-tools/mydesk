# mydesk

macOS 개인 환경 프로비저닝 및 설정 동기화 도구

## 기술 스택

- Go

## 시작하기

### 새 맥에서 부트스트랩

```bash
bash <(curl -sL https://raw.githubusercontent.com/silee9019/mydesk/main/bootstrap.sh)
```

### 개발

```bash
mise install       # Go 설치
mise run build     # 빌드
mise run test      # 테스트
mise run lint      # 린트
mise run fmt       # 포맷
```
